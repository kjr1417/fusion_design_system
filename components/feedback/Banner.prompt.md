Banner is a full-width, page- or section-level status message — not a toast (it doesn't auto-dismiss) and not per-field validation (use FormField for that).

```jsx
<Banner status="warning" variant="secondary">Markets close early on Nov 28.</Banner>
```

Four statuses map to the semantic colors; variant="secondary" is more attention-grabbing (tinted fill) than the default outlined "primary".
