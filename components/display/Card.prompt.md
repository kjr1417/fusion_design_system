Card groups related content in a bordered surface — a summary tile, list-row alternative, or dashboard module.

```jsx
<Card variant="primary" accent="left" hoverable>
  <h3>Checking · ...4821</h3>
  <p>$12,480.55</p>
</Card>
```

hoverable is for clickable cards (adds shadow + border highlight); leave it off for static display cards.
